## 使用說明
![說明]()

● 點選貓或狗後，接下來的題目會自動出現，依序點選回答，即可搜尋收容所的動物資料囉！

①在選單點選貓(左)或狗(右)

②傳送位置，系統回傳五個最近的收容所

③選擇收容所

④選擇動物性別

⑤選擇動物體型

⑥稍等一下，機器人在依照條件搜尋資料了~
